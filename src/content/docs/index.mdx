## What is DiSky?

DiSky is a comprehensive Skript addon that enables you to create and manage Discord bots entirely within Skript. No Java or complex programming knowledge required - if you know Skript, you can build Discord bots!

### Key Features

- **Easy to Use**: Familiar Skript syntax for Discord bot development
- **Comprehensive**: Support for messages, interactions, voice, modals, and more
- **Well Documented**: Extensive documentation with examples
- **Active Development**: Regular updates and new features
- **Community Support**: Active Discord community for help
