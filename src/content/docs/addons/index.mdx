Add-ons are specialized extensions that provide advanced features beyond Discord integration, perfect for enhancing your bot's capabilities.

## What are Add-ons?

While **Modules** extend DiSky's Discord functionality, **Add-ons** are separate plugins that provide additional features for your Minecraft server and bots:

- **Standalone Plugins**: Work alongside DiSky (not modules)
- **Specialized Features**: Image manipulation, book creation, and more
- **Full Integration**: Designed to work seamlessly with DiSky bots
- **Premium Quality**: Professional-grade functionality

<Aside type="caution" title="Premium Add-ons">
All DiSky add-ons are premium extensions available exclusively to [ItsTheSky's Patreon](https://www.patreon.com/itsthesky) supporters.
</Aside>

## Installation

Add-ons are installed like regular Minecraft plugins:

1. **Download**: Get the add-on JAR from [DiSky Resources](https://resources.disky.me/resources?premium) (requires active Patreon subscription)

2. **Install**: Place the JAR in your `/plugins/` folder (not `/plugins/DiSky/modules/`)

3. **Restart**: Restart your Minecraft server

4. **Verify**: Check console for successful load or use `/plugins` command

<Aside type="note">
Unlike modules, add-ons are **full plugins** that install in the main plugins folder, not the DiSky modules folder.
</Aside>

## Comparison: Modules vs Add-ons

| Feature | Modules | Add-ons |
|---------|---------|---------|
| **Installation** | `/plugins/DiSky/modules/` | `/plugins/` |
| **Type** | DiSky extensions | Standalone plugins |
| **Purpose** | Discord features | General Minecraft features |
| **Requires DiSky** | Yes | No (but designed for it) |
| **Examples** | LavaPlayer, OAuth, Webhooks | SkImage, BooSK |

## Troubleshooting

### Add-on Not Loading

- Verify JAR is in `/plugins/` (not `/plugins/DiSky/modules/`)
- Check server logs for error messages
- Ensure compatible Minecraft/Skript version
- Restart server after installation

### Missing Syntax

- Confirm add-on loaded successfully with `/plugins`
- Check Skript version compatibility
- Reload Skript scripts after installing add-on
- Review documentation for correct syntax

### Premium Access

- Verify active Patreon subscription
- Download from official [DiSky Resources](https://resources.disky.me/resources?premium)
- Join [DiSky Discord](https://disky.me/discord) for support
